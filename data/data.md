## Dataset Information

### Dataset Name
Fake and Real News Dataset

### Source
Kaggle: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

---

### Description
This dataset contains news articles labeled as fake or real news.  
It is used for binary text classification tasks in Natural Language Processing (NLP).

---

### Dataset Details
- Number of articles: ~20,000
- Task type: Binary classification (Fake vs Real News)

---

### Features
- `title` → News headline  
- `text` → Full news article content  

---

### Labels
- `0` → Fake News  
- `1` → Real News  

---

### Objective
The goal of this dataset is to build a machine learning model that can automatically detect fake news based on text content.
